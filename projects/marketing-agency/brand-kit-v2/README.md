## 🚀 Quick Start

### For Immediate Use:
1. **Logos**: Use files from `/logos/` - SVG for print/web, PNG for presentations
2. **Colors**: Reference `/html/palette.html` for exact hex values
3. **Typography**: Follow hierarchy in `/html/typography.html`
4. **Social**: Customize templates in `/social-templates/`

### For Website Implementation:
1. Upload favicon files to website root
2. Add favicon HTML snippet to `<head>` section  
3. Reference color and typography specs from HTML guides
4. Apply brand voice principles from guidelines

### For Design Work:
1. Import SVG logos into your design software
2. Use exact hex values from color palette
3. Install Google Fonts: Playfair Display, Inter Tight
4. Follow spacing and usage rules from guidelines

## 📐 Usage Guidelines

### Logo Do's
✅ Use approved logo files  
✅ Maintain clear space (height of "a" in "amotive")  
✅ Respect minimum sizes (16px web, 24px mobile, 48px print)  
✅ Use appropriate version for background color  

### Logo Don'ts  
❌ Stretch, distort, or rotate  
❌ Add effects, outlines, or drop shadows  
❌ Change colors or fonts  
❌ Use below minimum sizes  
❌ Place on busy backgrounds  

### Voice & Tone
✅ Direct, confident communication  
✅ Questions that make prospects think  
✅ Peer-to-peer business language  
✅ Results-focused messaging  

❌ Corporate jargon or buzzwords  
❌ Pushy or desperate language  
❌ Overly technical explanations  
❌ Generic marketing speak

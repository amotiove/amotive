Usage Guidelines

✓ Do

  • Use navy as primary text color on light backgrounds
  • Use gold sparingly for brand accents and highlights
  • Pair orange with white or light backgrounds for CTAs
  • Maintain consistent color relationships across touchpoints
  • Use light backgrounds to create visual hierarchy
  • Test contrast ratios for accessibility

✗ Don't

  • Use gold and orange together without white space
  • Place navy text on dark backgrounds
  • Overuse the gold - it should feel premium, not overwhelming
  • Use red and orange in the same composition
  • Create custom colors outside this palette
  • Use low-contrast combinations for body text

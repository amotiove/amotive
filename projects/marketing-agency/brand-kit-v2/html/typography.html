Best Practices

📱 Responsive Typography: All sizes scale down by 25% on mobile devices. H1 becomes 36px, H2 becomes 28px, etc.

✓ Do

  • • Use Playfair Display for brand personality
  • • Stick to Inter Tight for most UI text
  • • Maintain consistent line heights
  • • Use proper contrast ratios
  • • Scale typography for mobile

✗ Don't

  • • Mix more than 2 fonts in one design
  • • Use Playfair Display for long-form text
  • • Go below 14px for body text
  • • Use too many font weights
  • • Ignore mobile typography needs
